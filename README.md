# booking

## Описание
Веб-сервис для бронирования отелей

## Технологии
- GIT
### Backend
- Python
- FastAPI
- PostgreSQL
- Redis
- Celery
- Alembic
- JWT (Auth)
- SQLalchemy
### Frontend
- Typescript
- Vue.js
- Vite
- Quasar
### Deploying
- VPS (Ubuntu 20.04)
- S3 Storage
- Docker / Docker-compose
- Nginx
- TLS + domen
## Функционал
- Регистрация / Аутентификация / Авторизация пользователя
- Поиск отелей (возможен поиск по параметрам)
- Бронирование номеров
- Админ панель для управления CRUD
